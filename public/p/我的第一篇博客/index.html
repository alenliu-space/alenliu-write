我的第一篇博客

March 18, 2024

我想我第一篇POST需要霞与飞鸟 #

前言 #

这在实质意义上算作是我的第二篇博客，因为我的失误操作* n，我bomb这个web两次，导致我的第一篇博客也就此消失😭😭😭。磕磕绊绊，终于重新搭建好了一个较为粗略的博客（之后将不断更改😁）

溯因 #

一个幼稚但是值得我执着的想法 #

三月的第一个星期，因为众多公司都不招收暑期实习，且身为大一学生，本人的简历也略显单薄，加上HR在交流过程中的一些较为现实的话语😫，所以我想把我的相关摄影作品（严格意义上那并不能称之为摄影作品，只是一些粗涩的调色无构图物）放在一个网站上，这样即便我的短视频账号的浏览量并不引人，也能在一定程度上证明我或许具备拍摄较为高质量作品的潜力或者能力。但其实我并没有很好的去经营我的自媒体账号，更多的是将他们作为一个朋友圈去发表作品这一情况也是事实，所以我也无话可说啦哈哈哈。。。😮‍💨

想法——转化 #

同大多数人的初心不同，我并非一开始就想着搭建博客，毫无计算机基础的我当时只是单纯的想做一个Gallery网站以达到作品集的效果，然而，查找了相关资料且亲自动手之后才发现，Gallery远比博客的性价比要低许多，甚至我建立一个博客也是能够将Gallery网站放在我的Blog当中的，至此，我的博客搭建正式拉开帷幕….

Progress #

有如上述，我是一个计算机zero基础的人，不论是vscode、GitHub还是obsidian，前端三刺客，对我而言可以说是外星文明，一头雾水了很久（现在也依旧很苦恼就是了😥），一点点啃、一点点爬，最终终于能够基本运用了（虽然css和js还是看不太懂就是了😒） 但是，好在我有两个super friend，他们可以说是一步一步教一步一步讲了（真是辛苦他们了🤣），也终于，在磕磕绊绊的一个星期里，我完成了这项对我这个beginner来说的一项大工程———网站的构建和博客的书写 …..但是这似乎离我的Gallery还有超级长的一段距离😫，还有好多内容在那里等着我 可是能够这样的一步步学一步步做的感觉，不赖

落墨–终 #

今日天气很好，好到我不想盯着这块屏幕，不想看这堆让我畏惧又让我兴奋的代码 最终我还是坐下来了，吹着穿过宿舍的风，喝着氤氲的茶，写下这篇没什么文采的不出意外将会真正意义上成为我第一篇博客的文章

春分之临，予一张春天与君共享

希望我的博客会有更好的样子，希望大家在生活中也能有更好的样子。
